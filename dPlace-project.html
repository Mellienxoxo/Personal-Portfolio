D'Place Website

D'Place is a website dedicated for a restaurent that sells various dishes. It's purpose is to connect and build strong relationship to the customer and also to build the business brand.

This website utilize Wordpress and Elementor page builder. It also used HTML, CSS, JavaScript for the front-end, and PHP for the back-end. It also apply various plugins to extend the functionalities, boost the performance and to secure the data and also the website itself.
