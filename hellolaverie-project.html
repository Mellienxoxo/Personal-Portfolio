Lavpos Web-based System

It is a Point of Sales with Sales Monitoring System that dedicated for Hello Laverie Laundry Shop. Adding to that, this system is also my undergraduate thesis in my College years.

The web application uses CodeIgniter for the framework, PHP for the back-end, HTML, CSS, BOOTSTRAP, and JavaScript for the front-end. It also use MySql for the database. Furthermore, the web-base system also utilized Semaphore API for SMS Gateway

Type: Business
